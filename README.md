# Simple ATM WEB API
## API Docs
You can use Swagger via path `/swagger_doc` to check your documentation.
It is very easy via [Swagger UI](https://github.com/swagger-api/swagger-ui)
## Routes

### GET /api/max_withdraw
### POST /api/load
### POST /api/withdraw


## Run
Using Rack
```
bundle install
rackup -p 8080
```

Using Docker Compose
```
docker-compose up
```
